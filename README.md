# My App
A React Native tutorial app for learning about buttons and React Native components.

## About
This project is part of a React Native tutorial series. In this chapter, we learn about buttons and how to implement interactive UI elements in React Native.

## Prerequisites
- Node.js >= 20
- React Native development environment set up
- Android Studio (for Android development)
- Xcode (for iOS development, macOS only)

## Installation
1. Install dependencies:
```bash
npm install
```

2. For iOS (macOS only):
```bash
cd ios && pod install && cd ..
```

## Running the App

### Start Metro Bundler
```bash
npm start
```

### Run on Android
```bash
npm run android
```

### Run on iOS
```bash
npm run ios
```

## Project Structure
- `App.tsx` - Main application component with button examples
- `index.js` - Application entry point
- `android/` - Android native code
- `ios/` - iOS native code

## Features
- Basic button implementation
- Button press event handling
- Simple alert interactions

## Scripts
- `npm start` - Start Metro bundler
- `npm run android` - Run on Android device/emulator
- `npm run ios` - Run on iOS device/simulator
- `npm test` - Run tests
- `npm run lint` - Run ESLint
